Run this command to port forward to the Grafana pod in app-03:                                                                     

                                                                                                                                   
 kubectl port-forward -n app-03 pod/grafana-7f8b9c6d5-x2m4n 3000:3000                                                              
                                                                                                                                   

This forwards local port 3000 to port 3000 on the pod.
