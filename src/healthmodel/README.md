# az healthmodel

Azure CLI extension for the Microsoft.CloudHealth resource provider (API version `2026-01-01-preview`).

## Install (local dev)

```bash
pip install -e /path/to/azure-mgmt-cloudhealth
pip install -e .
```

## Commands

```
az healthmodel   create|show|list|update|delete
az healthmodel entity         create|show|list|delete|get-history|get-signal-history|ingest
az healthmodel signal-definition  create|show|list|delete
az healthmodel relationship   create|show|list|delete
az healthmodel auth-setting   create|show|list|delete
az healthmodel discovery-rule create|show|list|delete
```
