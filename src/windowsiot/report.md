# Azure CLI Module Creation Report

### windowsiotservices service check-device-service-name-availability

check-device-service-name-availability a windowsiotservices service.

|Option|Type|Description|Path (SDK)|Path (swagger)|
|------|----|-----------|----------|--------------|
|**--name**|string|The name of the Windows IoT Device Service to check.|name|
### windowsiotservices service create

create a windowsiotservices service.

|Option|Type|Description|Path (SDK)|Path (swagger)|
|------|----|-----------|----------|--------------|
|**--resource-group-name**|string|The name of the resource group that contains the Windows IoT Device Service.|resource_group_name|
|**--device-name**|string|The name of the Windows IoT Device Service.|device_name|
|**--if-match**|string|ETag of the Windows IoT Device Service. Do not specify for creating a new Windows IoT Device Service. Required to update an existing Windows IoT Device Service.|if_match|
|**--notes**|string|Windows IoT Device Service notes.|notes|
|**--quantity**|integer|Windows IoT Device Service device allocation,|quantity|
|**--billing-domain-name**|string|Windows IoT Device Service ODM AAD domain|billing_domain_name|
|**--admin-domain-name**|string|Windows IoT Device Service OEM AAD domain|admin_domain_name|
