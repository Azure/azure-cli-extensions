# Azure CLI Discovery Extension #
This is an extension to Azure CLI to manage Microsoft Discovery resources.

## How to use ##
Install the extension:
```
az extension add --name discovery
```

### Commands ###
```
az discovery bookshelf       : Manage bookshelves.
az discovery storage-container : Manage storage containers.
az discovery supercomputer   : Manage supercomputers.
az discovery tool            : Manage tools.
az discovery workspace       : Manage workspaces.
```
